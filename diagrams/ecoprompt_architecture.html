EcoPrompt Architecture

flowchart TB
    User["User"] --> UI["React Frontend
Vite App"]

    UI --> ChatUI["Chat Console
messages, history, web toggle, copy buttons"]
    UI --> DashUI["Analytics Dashboard
latency, cost, energy, route distribution"]

    ChatUI --> API["FastAPI Backend
main.py"]
    DashUI --> MetricsAPI["GET /metrics"]
    MetricsAPI --> MetricsStore["In-Memory Metrics Store"]

    API --> Validate["Input Validation"]

    Validate --> Deterministic["Deterministic Engine
math, conversions, greetings, prime checks"]
    Deterministic -->|matched| Response["Response to Frontend"]

    Validate -->|web_search = true| WebRoute["Web Search Route"]
    WebRoute --> Gemini["Gemini API
Google Search Grounding"]
    Gemini --> SourceFilter["Source Extraction + Trust Filter
block weak domains, rank trusted sources"]
    SourceFilter --> LocalFormatter["Optional Local Formatting
Ollama llama3.2:1b"]
    LocalFormatter --> Response

    Validate -->|code request| TemplateEngine["Code Template Engine
kb/code_templates.json"]
    TemplateEngine -->|template match| Response
    TemplateEngine -->|no template| LocalCode["Local Model Code Fallback
normalize into fenced code block"]
    LocalCode --> Response

    Validate --> KBLookup["Domain KB Lookups"]
    KBLookup --> Geography["Geography KB"]
    KBLookup --> MathKB["Math KB"]
    KBLookup --> Science["Science KB"]
    KBLookup --> History["History KB"]
    KBLookup --> Programming["Programming KB"]
    KBLookup --> HighLevel["High-Level Concepts KB"]

    KBLookup -->|relevant match| KBContext["Build Context Prompt"]
    KBContext --> LocalModel["Local Ollama Model
llama3.2:1b"]
    LocalModel --> Response

    Validate --> RAG["Local RAG Engine
retrieve_top_k"]
    RAG --> LocalJSON["Local JSON Knowledge Files"]
    LocalJSON --> RAGContext["Build RAG Context Prompt"]
    RAGContext --> LocalModel

    Validate -->|fallback| LocalModel
    LocalModel --> WeakCheck["Weak Response / Complexity Check"]

    WeakCheck -->|simple or good enough| Response
    WeakCheck -->|complex or weak non-web prompt| Groq["Groq Fallback
llama-3.1-8b-instant"]
    Groq --> Response

    Response --> UI

    Response --> MetricsUpdate["Update Metrics
route, latency, energy, cost"]
    MetricsUpdate --> MetricsStore

    ChatUI --> BrowserStorage["Browser localStorage
chat history + titles"]
    BrowserStorage --> ChatUI
